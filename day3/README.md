#### 回文诗

回文诗，也写作“爱情诗”“回环诗”。它是汉语特有的一种使用词序回环往复的修辞方法，文体上称之为“回文体”。

它有多种表现形式，如唐.潘孟阳春日诗是把诗，从最后一个字起倒读又出了另外一首诗了。这类表现形式的诗题材多。这种回文诗叫大回。

例如：
正读
春梅杂落雪 发树几花开 真须尽兴饮 仁里愿同来
倒读
来同愿里仁 饮兴尽须真 开花几数发 雪落杂梅春
